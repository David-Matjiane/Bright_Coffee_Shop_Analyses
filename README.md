# Bright_Coffee_Shop_Analyses

# Project Description
## 📌 Overview
This project analyzes sales data from Bright Coffee Shop to uncover key business insights. 
The goal is to identify top-performing products, peak sales periods, and provide 
data-driven recommendations to improve revenue.

## 🎯 Objectives
- Identify top-selling products
- Analyze sales trends over time
- Determine peak business hours
- Provide actionable business recommendations

  ## 🛠 Tools Used
- Databricks (Data Processing)
- SQL (Data Analysis)
- Microsoft Excel (Data Visualization)
- Power BI / Tableau (Optional Dashboards)
- Miro (Planning & Architecture)

  ## 📂 Project Structure
- /miro → Planning diagram
- /Canva → Planning gannt chart
- /data → Raw and processed datasets
- /sql → SQL queries used for analysis
- /visuals → Charts and dashboards
- /presentation → Final PowerPoint presentation

 ## ⚙️ Data Processing
- Converted Excel data to CSV
- Loaded data into Databricks
- Cleaned and transformed data
- Created new columns (e.g., time buckets, total_amount)
- Aggregated data using SQL

  ## 📊 Key Insights
- Revenue increased from February to June
- Morning sales contribute over 50% of revenue
- Coffee is the top-performing product category
- Evening sales are the lowest

  ## 💡 Recommendations
- Focus marketing efforts on evening sales
- Promote best-selling products
- Introduce bundle deals (coffee + bakery)
